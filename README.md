# Super-Store-Sales-Analysis

1. Project Overview

This project presents a comprehensive Sales Analysis Dashboard built using Power BI, aimed at extracting actionable insights from retail sales data. It helps stakeholders understand sales trends, customer behavior, and business performance to support data-driven decision-making.

2. Objectives

Analyze overall sales performance across regions and categories
Identify top-performing products and customer segments
Track profit trends and detect loss-making areas
Provide interactive visualizations for better business insights.

3. Tools & Technologies

Power BI (Data Visualization & Dashboarding)
Excel / CSV Dataset (Data Source)
Data Cleaning & Transformation (Power Query).

4. Key Features

 Region-wise Sales Analysis
 Category & Sub-category Performance
 Profit vs Sales Trend Visualization
 Customer Segment Insights
 Time-based Sales Trends (Monthly/Yearly)
 Interactive Filters & Slicers

5. Insights Derived
 
Identified high-revenue generating regions and products
Detected categories with consistent losses
Observed seasonal trends in sales performance
Highlighted customer segments contributing most to profit.
